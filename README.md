# Simple online shop 

This is a sample project to learn VueJS 

Include :
* [Bootstrap 5](https://getbootstrap.com)
* [Vue 3.2](https://vuejs.org)
* [Vue router](https://router.vuejs.org/)
* [Vite](https://vitejs.dev/)
* [Pinia](https://pinia.vuejs.org/)

## How to use

```sh
npm install
npm run build
npm run start
```
